# pi-open-message-in-browser (monorepo)


[Markdown preview extension](https://github.com/simov/markdown-viewer) repackaged as cli and [pi](https://github.com/earendil-works/pi) extension. Also it has mermaid support and validation

## Install

- As a `pi` extension: `just install-extension`
- As a standalone CLI: `just install`

## Development

```bash
bun install               # link workspaces
bun run --filter mdopen build
```
